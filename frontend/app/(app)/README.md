# frontend/app/(app) Folder Guide

## Purpose
Authenticated clinician workspace shell and routed pages.

## Subfolders
| Folder | Role |
|---|---|
| `assistant` | Doctor-facing chatbot workspace for contextual guidance. |
| `assessments` | Assessment route group for Stage 1, Stage 2, and Stage 3 screens. |
| `patients` | Patient list/create/delete and active patient controls. |
| `reports` | Report generation and PDF retrieval page. |

## Files
| File | What it does |
|---|---|
| `layout.tsx` | Next.js layout wrapper for nested routes. |
