# ml/src Folder Guide

## Purpose
Shared ML library code for data handling, model building, metrics, and Stage 1 explainability.

## Files
| File | What it does |
|---|---|
| `__init__.py` | Package marker and module export surface. |
| `data.py` | Python module implementing domain-specific logic for this folder. |
| `metrics.py` | Model metric computation and quality gate helpers. |
| `modeling.py` | Model registry/status or model lifecycle helper. |
| `stage1_data.py` | Stage 1 clinical scoring or model support module. |
| `stage1_explainability.py` | Stage 1 clinical scoring or model support module. |
| `stage1_modeling.py` | Stage 1 clinical scoring or model support module. |
