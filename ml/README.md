# ml Folder Guide

## Purpose
ML training/evaluation workspace for Stage 1, Stage 2 support, and Stage 3 synthetic/risk models.

## Subfolders
| Folder | Role |
|---|---|
| `artifacts` | Persisted model artifacts and metadata generated by ML pipelines. |
| `configs` | YAML training/evaluation configurations. |
| `scripts` | Executable pipelines for training, evaluation, drift checks, synthetic generation, and registry updates. |
| `src` | Shared ML library code for data handling, model building, metrics, and Stage 1 explainability. |
| `tests` | ML contract and pipeline tests. |

## Files
| File | What it does |
|---|---|
| `requirements.txt` | Project file used by this folder. |

## Quick Commands
- `python3 -m pip install -r requirements.txt`
- `python3 scripts/train_stage1.py --config configs/train_stage1.yaml`
- `python3 scripts/generate_stage3_synthetic.py --patients 250000 --visits 12`
- `python3 scripts/train_stage3.py --config configs/train_stage3.yaml`
